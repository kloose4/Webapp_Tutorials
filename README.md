<h1>Meine erste Webapp</h1>

<h2>Repository klonen:</h2>
<ul>Erstelle einen neuen, leeren Ordner und öffne diesen mit Visual Studio Code.</ul>
<ul>Öffne in Visual Studio Code ein neues Terminal.</ul>
<ul>"git clone https://github.com/kloose4/Webapp_Tutorials.git"</ul>

<h2>Flask installieren:</h2>
<ul>Öffne auf der Festplatte C:\ eine Eingabeaufforderung (Du kannst nach "cmd64.exe" suchen).</ul>
<ul>Befehl "cd C:\Program Files\Python314\Scripts"</ul>
<ul>Befehl "pip install Flask"</ul>
